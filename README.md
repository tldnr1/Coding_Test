## 코딩테스트 준비 (https://www.acmicpc.net/problem/tags)
* 브루트 포스
* 자료 구조 ( 배열, 연결 리스트, 스택, 큐, 우선순위 큐, 트리, 해시 등등등... )
* 정렬 (단독으로는 잘 안나오고, 그리디나 이분탐색 등 다른 것들과 엮어서)
* 시뮬레이션 / 그리디 / DP (Knapsack, LCS, LIS) / BFS / DFS / 백트래킹 / 재귀
* 슬라이딩 윈도우 / 투 포인터
* 이분 탐색 + 파라메트릭 서치
* 최단 경로 (다익스트라 & 플로이드 워셜)
* (여유 되면)
    * 최소 신장 트리 & 유니온 파인드
    * KMP
    * 트라이
    * LCA
